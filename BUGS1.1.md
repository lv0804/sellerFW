
# Bug Report: Страница списка объявлений (котята)

## 1. Некорректная логика кнопки «Добавить в избранное»

**ID:** BUG-001
**Приоритет:** P1 — высокий
**Серьёзность:** Major

**Описание:**
Кнопка «сердечко» (добавление в избранное) работает некорректно: состояние не соответствует фактическому добавлению.

**Шаги для воспроизведения:**

1. Открыть страницу объявлений
2. Нажать на иконку сердца у любого объявления
3. Обновить страницу или проверить список избранного

**Фактический результат:**

* Иконка может менять состояние без сохранения
* Отображается сообщение "больше на показывать"
* Либо состояние не соответствует факту добавления

**Ожидаемый результат:**

* Состояние кнопки синхронизировано с избранным
* Данные сохраняются после обновления страницы
* Сообщание "Добавлено в избранное"

## 2. Некорректная работа сортировки по городам

**ID:** BUG-002
**Приоритет:** P1 — высокий
**Серьёзность:** Major

**Описание:**
Сортировка/фильтрация по городам работает неправильно (например, выбран один город, но отображаются объявления из других).

**Шаги:**

1. Выбрать город (например, Москва)
2. Применить фильтр/сортировку

**Фактический результат:**
Отображаются объявления из других городов (например, Новосибирск)

**Ожидаемый результат:**
Показываются объявления только из выбранного города
