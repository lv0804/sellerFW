# TESTCASES.md

Ниже перечислены тест-кейсы для задания 2.2 по UI платформы модерации объявлений.

## Общие предусловия

- Стенд: `https://cerulean-praline-8e5aa6.netlify.app`
- Браузер десктоп: Chrome/Chromium, viewport `1440x1200`
- Браузер мобильный: viewport `390x844`
- Данные на стенде изменяемые, поэтому проверки не опираются на фиксированные ID, названия и количество объявлений

## Десктоп

### TC-UI-001. Главная страница: фильтр `Диапазон цен`

- Приоритет: High
- Шаги:
  1. Открыть главную страницу.
  2. В поле `От` ввести `10000`.
  3. В поле `До` ввести `50000`.
- Ожидаемый результат:
  - Список объявлений успешно обновился.
  - На странице отображаются объявления.
  - Цена каждого видимого объявления находится в диапазоне `10000-50000`.

### TC-UI-002. Главная страница: сортировка `По цене`

- Приоритет: High
- Шаги:
  1. Открыть главную страницу.
  2. В поле `Сортировать по` выбрать `Цене`.
  3. В поле `Порядок` выбрать `По возрастанию`.
  4. Проверить порядок цен у видимых объявлений.
  5. Переключить `Порядок` на `По убыванию`.
  6. Повторно проверить порядок цен.
- Ожидаемый результат:
  - При порядке `По возрастанию` цены упорядочены по возрастанию.
  - При порядке `По убыванию` цены упорядочены по убыванию.

### TC-UI-003. Главная страница: фильтр `Категория`

- Ожидаемый результат:

  - На странице отображаются объявления.
  - Каждое видимое объявление относится к категории `Электроника`.

### TC-UI-005. Страница статистики: кнопка `Обновить`

- Приоритет: High
- Шаги:
  1. Открыть страницу статистики через навигацию приложения.
  2. Дождаться отображения таймера автообновления.
  3. Подождать 2-3 секунды и убедиться, что значение таймера уменьшилось.
  4. Нажать кнопку `Обновить`.
- Ожидаемый результат:
  - Таймер отображается до нажатия кнопки.
  - После нажатия `Обновить` таймер перезапускается и его значение увеличивается относительно текущего.

### TC-UI-006. Страница статистики: кнопка остановки таймера

- Приоритет: High

  2. Нажать кнопку остановки таймера.
- Ожидаемый результат:
  - Вместо времени отображается сообщение `Автообновление выключено`.
  - Кнопка запуска таймера становится доступной.

### TC-UI-007. Страница статистики: кнопка запуска таймера

- Приоритет: High
- Шаги:
  1. Открыть страницу статистики через навигацию приложения.
  2. Остановить таймер.
  3. Нажать кнопку запуска таймера.
- Ожидаемый результат:
  - Сообщение `Автообновление выключено` исчезает.
  - Снова отображается таймер автообновления.
  - Таймер начинает отсчет.

## Мобильная версия

### TC-UI-008. Переключение между тёмной и светлой темой

- Приоритет: Medium
- Шаги:
  1. Открыть главную страницу на мобильном viewport.
  2. Нажать кнопку смены темы.
  3. Зафиксировать изменение визуального состояния страницы.
  4. Нажать кнопку повторно.
- Ожидаемый результат:
  - После первого нажатия интерфейс переключается в противоположную тему.
  - После второго нажатия интерфейс возвращается в исходную тему.
  - Иконка тогла темы меняется синхронно с темой.
