
## BUG-UI-001. Фильтр `Только срочные` зависает в состоянии загрузки

- Severity: High
- Среда: desktop, `https://cerulean-praline-8e5aa6.netlify.app`
- Шаги:
  1. Открыть главную страницу.
  2. Включить тогл `Только срочные`.
- Ожидаемый результат:
  - Список объявлений обновляется.
  - На странице остаются только объявления с меткой `Срочно`.
- Фактический результат:
  - После переключения на странице остается состояние `Загрузка объявлений...`.
  - Обновленный список объявлений не отображается.

## BUG-UI-002. Кнопка запуска таймера не возвращает автообновление статистики

- Severity: High
- Среда: desktop, страница статистики
- Шаги:
  1. Открыть страницу статистики через навигацию приложения.
  2. Нажать кнопку остановки таймера.
  3. Нажать кнопку запуска таймера.
- Ожидаемый результат:
  - Сообщение `Автообновление выключено` исчезает.
  - Снова отображается таймер автообновления.
  - Таймер начинает отсчет.
- Фактический результат:
  - После нажатия кнопки запуска состояние `Автообновление выключено` сохраняется.
  - Таймер не возвращается.

## BUG-UI-003. Прямое открытие маршрутов `/list` и `/stats` возвращает 404

- Severity: Medium
- Среда: desktop/mobile, прямой переход по URL
- Шаги:
  1. Открыть `https://cerulean-praline-8e5aa6.netlify.app/list`.
  2. Открыть `https://cerulean-praline-8e5aa6.netlify.app/stats`.
- Ожидаемый результат:
  - Открывается соответствующая страница SPA.
- Фактический результат:
  - Netlify отдает `Page not found`.
  - Страницы доступны только через внутреннюю навигацию из `/`.
